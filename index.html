welcome to the pixel language homepage

welcome to pixel. this project consists of a python/lisp based programming language and its compiler. the intention is help other people to know more of how to create a programming language, and learn contributing, so, at the same time, a new technology is being created.

changes

primarily, the project was built with java. we are rewriting it to c++ (due to performance issues), and changing the language syntax a little bit. i've got into a good approach for pixel discussing with other folks on hacker news, in which the syntax will be changed to a python like syntax but with lisp like operators. i thought it really interesting, due to the fact that its more easy to be implemented and also it will have a better readable property. please tell me what do you think about this new approach. here is an example of how the syntax will look like. some changes will happen yet, but this is the start.

def main []
    int a1 a2 a3
    str s1 s2
    = a1 10
    = a2 a1
    = a1 (- a2 10)
    = s1 "house"
    real r1 r2 r3
    = r1 (+ (- r3 r2) r3)
    = r2 10.25

    if > a1 a2
        = a3 2
    else
        = a3 29
        = a2 10

    while <= a3 100
        = a3 (+ a3 2)
        = a1 100

    read a1
    read a2
    print "final test"

source code | contribute

there is a stable version in java which consists of the compiler and the analyzers. the source code is available here. however, we are reimplementing it to c++, so we are rewriting it based on the java implementation.

if you want to contribute to this project, clone the source code on github and read the implementation requirements before you start coding.
